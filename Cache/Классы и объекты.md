## Классы и объекты
- **Класс** — шаблон для создания объектов. Определяется через `class`.
- **Объект** — экземпляр класса. Создается через конструктор: `obj = ClassName()`.
- **[[Атрибуты]]** — переменные внутри класса (например, `self.name`).
- **[[Методы]]** — функции внутри класса. Первый параметр — `self`.
- **Конструктор `__init__`** — инициализирует объект: 

  ```python
  class Person:
      def __init__(self, name):
          self.name = name
  ```
