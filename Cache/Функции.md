## Функции
- Определяются через `def`:  

  ```python
  def sum(a, b):
      return a + b
  ```

- **[[Аргументы|Аргументы]]**: позиционные, именованные, значения по умолчанию.
- **`*args`** и **`**kwargs`** для произвольного числа аргументов.
- **Лямбда-функции**:  

  ```python
  multiply = lambda x, y: x * y
  ```
  
- Связи: [[Рекурсия]].
