Нефункциональное тестирование — это тип тестирования, который фокусируется на аспектах работы системы, не связанных с её функциональностью.

Некоторые виды нефункционального тестирования:

| **Вид**                                       | **Описание**                                                                 | **Пример**                                                                 |
| --------------------------------------------- | ---------------------------------------------------------------------------- | -------------------------------------------------------------------------- |
| [[Юзабилити-тестирование]]                    | Оценка удобства использования интерфейса                                     | Тестирование навигации для новых пользователей                             |
| [[Стресс-тестирование]]                       | Проверка работы в экстремальных условиях                                     | Работа системы при 200% от плановой нагрузки                               |
| [[Нагрузочное тестирование]]                  | Определение производительности под нагрузкой                                 | Одновременная работа 1000 пользователей                                    |
| [[Объёмное тестирование]]                     | Обработка больших объёмов данных                                             | Импорт базы данных с 1 млн записей                                         |
| [[Инсталляционное тестирование]]              | Проверка процессов установки/обновления                                      | Корректность обновления с версии 2.0 до 3.0                                |
| [[Тестирование на отказ и восстановление]]    | Проверка реакции на сбои и восстановления                                    | Отключение сервера БД во время операции                                    |
| [[Конфигурационное тестирование]]             | Проверка работы в разных окружениях                                          | Запуск на Windows/Linux, разных версиях iOS/Android                        |
| [[Тестирование совместимости]]                | Проверка взаимодействия с ОС, браузерами, устройствами                       | Работа веб-приложения в Chrome, Firefox, Safari                            |
| [[Эксплуатационное тестирование]]             | Проверка в условиях, близких к реальной эксплуатации                         | Анализ работы почтового сервиса при массовой отправке писем                |
