> [!info] Определение
> Адаптивный дизайн - Это способ корректно отображать интерфейс сайта на любых устройствах. Дизайнер отрисовывает отдельные макеты веб-страниц для смартфонов, планшетов и компьютеров. Разработчик при вёрстке задаёт для каждого макета соответствующие размеры и разрешение экрана. Когда пользователь заходит на сайт, система распознаёт тип его устройства и показывает подходящий вид интерфейса.

---
## Особенности адаптивного дизайна

- Количество макетов может варьироваться (от двух до шести и более) в зависимости от сложности интерфейса.

- На маленьких экранах элементы не просто уменьшаются, а меняется их расположение для удобства восприятия.

- Менее значимые элементы скрываются (например, в контекстное меню).

- Функциональность обычно идентична на всех устройствах, но иногда может быть упрощена для мобильных версий.
## Альтернативные подходы

### [[Респонсивный дизайн]]

- Сайт автоматически подстраивается под размер экрана с помощью программного модуля.

- **Плюсы:** проще и дешевле в разработке.

- **Минусы:** больше вес, дольше загрузка, не подходит для сложной вёрстки.

### [[Мобильная версия сайта]]

- Отдельный сайт со своим адресом, оптимизированный для мобильных устройств.

- **Плюсы:** может кардинально отличаться по дизайну и функциональности.

- **Минусы:** дороже в разработке, может путать пользователей из-за отдельного адреса.
## Преимущества адаптивного дизайна

- Оптимален по соотношению цены и возможностей.

- Подходит для нестандартных дизайн-решений.

- Обеспечивает быструю загрузку страниц.

- Учитывается поисковиками при ранжировании.
