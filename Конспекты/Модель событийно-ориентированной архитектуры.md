# Событийно-ориентированная архитектура

**Описание**:  
Компоненты системы реагируют на события (например, нажатие кнопки, изменение данных) через асинхронные сообщения.  
**Применение**:  
Торговые платформы, IoT-системы.  

> [!tip]
> **Преимущества**:
> 
> - Обработка данных в реальном времени.
>     
> - Высокая отказоустойчивость.
