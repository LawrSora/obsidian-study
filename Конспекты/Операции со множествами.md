### Объединение множеств

Объединение множеств (A, B) - [[множество]], состоящее из элементов принадлежащих хотя бы одному из этих множеств.
$$
A \lor B = \{x:x \in A \lor x \in B\}
$$
![[roundOr.jpg]]

Для операции объединения справедливы следующие законы:

- $A \lor A = A$ - закон идемпотентности.
- $A \lor B = B \lor A$ - коммутативный закон.
- $A \lor (B \lor C) = (A \lor B) \lor C$ - ассоциативный закон.
- $A \lor \varnothing = A$ - закон нейтральности.
- $A \lor U = U$ - закон подавления.

Если у множества отсутствуют элементы, то оно называется пустое множество $\varnothing$.

Универсальное множество - это множество, объединяющее элементы данного свойства.

### Пересечение множеств

Пересечение множеств - множество $A \land B$, состоящее из всех общих элементов этих множеств.
$$
A \land B = \{x:x\in A \land x \in B\}
$$
![[round&.jpg]]

Для операции пересечения справедливы следующие законы:

- $A \land A = A$ - закон идемпотентности.
- $A \land B = B \land A$ - коммутативный закон.
- $A \land (B \land C) = (A \land B) \land C$ - ассоциативный закон.
- $A \land U = A$ - закон нейтральности.
- $A \land \varnothing = \varnothing$ - закон подавления.

### Разность множеств

Разность множеств (A, B) – множество, состоящее из элементов, принадлежащих множеству A, но не принадлежащих множеству B.
$$
A \setminus B = \{x : x \in A \land x \notin B\}
$$
![[round-.jpg]]
Для операции разности справедливы следующие законы:

- $A \setminus A = \varnothing$ - закон идемпотентности.
- $A \setminus B \neq B \setminus A$ - отсутствие коммутативности.
- $A \setminus (B \lor C) = (A \setminus B) \land (A \setminus C)$ - закон распределения.
- $A \setminus \varnothing = A$ - закон нейтральности.

### Отрицание множеств

Отрицание множества $A$ - это множество, состоящее из элементов, которые не принадлежат множеству $A$. Обозначается как $\neg A$ (комплемент множества $A$). 

Формально, отрицание множества $A$ можно записать следующим образом:
$$
\neg A = \{x: x \in U \land x \notin A\}
$$
![[slide_16.jpg]]
Для операции отрицания справедливы следующие законы:

- $\neg A = A^c$ - определение комплемента.
- $\neg(\neg A) = A$ - закон двойного отрицания.
- $\neg(A \lor B) = \neg A \land \neg B$ - закон де Моргана (для объединения).
- $\neg(A \land B) = \neg A \lor \neg B$ - закон де Моргана (для пересечения).
- $\varnothing = U; \neg U = \varnothing$

### Симметрическая разность

Симметрическая разность - это множество $A \Delta B$, состоящее из элементов, принадлежащих тому или другому из этих множеств, но не обоим сразу.
$$
A \Delta B = \{x:x \in A \lor x \in B \land x \in A \land B\}
$$
![[roundxor 1.jpg]]

### Стрелка Пирса

Истинно только тогда, когда оба выражения ложны(отрицание дизъюнкции).
$\downarrow$

### Штрих Шеффера

Ложно только тогда, когда оба выражения истинны(отрицание конъюнкции)
$\mid$

### Сложение по модулю 2
(либо, либо)

Истинно только тогда, когда оба выражения не равны друг другу.
$\oplus$
