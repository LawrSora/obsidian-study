CSS (Cascading Style Sheets) — это язык стилей, используемый для описания внешнего вида и форматирования документов, написанных на [[HTML]] или XML. CSS позволяет отделить содержание [[Веб-страница|веб-страницы]] от её оформления, что делает код более организованным и легким для изменения.

Краткое описание:

CSS предоставляет набор правил для стилизации элементов на веб-странице, таких как шрифты, цвета, размеры, отступы, расположение и адаптивность [[Веб-дизайн|дизайна]]. Стили могут применяться к отдельным элементам, групповым селекторам или всему документу. CSS поддерживает каскадность, что означает, что стили могут наследоваться от родительских элементов или перекрываться более специфичными правилами, позволяя создавать гибкие и многоуровневые дизайны. Также CSS включает медиазапросы для создания адаптивных интерфейсов, которые хорошо выглядят на разных устройствах — от настольных компьютеров до мобильных телефонов.
