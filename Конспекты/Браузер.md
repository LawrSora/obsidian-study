Браузер — это программа для просмотра [[Веб-страница|веб-страниц]]. Она преобразует получаемый из сети код в понятные для человеческого восприятия элементы и позволяет управлять ими.

Основная функция браузера — открывать [[Веб-сайт|сайты]]. Для этого в интерфейсе каждой программы есть адресная строка — поле, в котором указывают адрес нужного сайта или поисковой запрос.
