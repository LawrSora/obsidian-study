# Как подобрать цвета для сайта или веб-приложения

> [!question] Почему цвета так важны
> Цвета помогают установить сильную **эмоциональную связь** с клиентами. Они передают определенные чувства и ассоциации, которые формируют восприятие бренда.

---
## Этапы подбора цветовой палитры

### 1. Определите характер вашего бренда
Какие эмоции и качества вы хотите передать? (Например: дружелюбие, надёжность, креативность, роскошь).

### 2. Изучите значение цветов
- **Красный**: Страсть, волнение, решительность. Привлекает внимание.
- **Оранжевый**: Игривость, жизненная сила, дружелюбие. Для молодой аудитории.
- **Желтый**: Счастье, молодость, оптимизм.
- **Зеленый**: Стабильность, природа, процветание. Тёмные оттенки — деньги.
- **Светло-голубой**: Спокойствие, доверие, открытость.
- **Тёмно-синий**: Профессионализм, безопасность. Популярен в финансах и IT.
- **Фиолетовый**: Креативность, роскошь, богатство.
- **Розовый**: Женственность, молодость, невинность.
- **Коричневый**: Консерватизм, опыт, надёжность. Используется редко.
- **Белый**: Чистота, здоровье, простота.
- **Чёрный и серый**: Роскошь, элегантность, стиль. Для премиум-сегмента.

### 3. Используйте формулу выбора цветов
Выберите три основных типа цветов:
1.  **Базовый цвет** — основной цвет бренда.
2.  **Акцентный цвет** — для кнопок, важных элементов.
3.  **Нейтральный цвет** — для фона, текста (часто серый, белый, бежевый).

### 4. Выберите тип цветовой схемы
- **Монохроматическая**: разные оттенки одного цвета.
- **Аналогичная**: несколько соседних цветов в круге.
- **Комплементарная**: противоположные цвета.
- **Триадная**: три цвета, равноудалённые в цветовом круге.

## Вывод
Цвета — ключевая часть визуальной идентичности. Они определяют восприятие сайта, логотипа и рекламы. Будьте последовательны и обоснованы в выборе палитры.
