# Конспект по теме "Базы данных"

База данных (БД) — это организованная структура для хранения, управления и извлечения [[Данные|данных]]. Она позволяет эффективно обрабатывать большие объемы информации.
## Типы баз данных

1. [[Реляционные базы данных]] (РБД)
2. [[Нереляционные базы данных]] (NoSQL)
3. [[Объектные базы данных]]
## Основные компоненты базы данных

- **Таблицы** — основная структура хранения данных в реляционных БД.
- **Записи** — строки в таблице, представляющие собой отдельные элементы данных.
- **Поля** — столбцы в таблице, определяющие атрибуты записей.
- **Индексы** — структуры, которые ускоряют доступ к данным.
## Операции с базой данных

- **Создание** — добавление новых таблиц и записей.
- **Чтение** — получение данных с помощью запросов.
- **Обновление** — изменение существующих данных.
- **Удаление** — удаление данных из базы.
